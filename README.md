# 🌐 Frontend Templates Collection

A collection of modern, responsive frontend templates built using **HTML, CSS, JavaScript, and React**.

These templates are designed for real-world business use cases and include features like **contact forms, booking systems, and email integration using EmailJS**.

---

## 🌍 Live Demos

* ☕ **Coffee Shop Template (React)**
  👉 https://the-coffee-haven.netlify.app

* ✂️ **Salon Booking Template**
  👉 https://elitesalonbarking.netlify.app/

---

## 🚀 Features

* ✅ Responsive and mobile-friendly design
